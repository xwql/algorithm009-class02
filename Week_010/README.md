毕业学习总结

在超哥的带领下，两个月来刷了一百多道题，虽然有点少，但是加上```五毒神掌```，自己在算法上面确实有了明显的进步，记住了各类算法的时间复杂度，从最基本的数据结构逐渐到高级的算法，一步一步把基本功打扎实，虽然在中高级算法方面我还有待多多刷题过遍数。

从刷题技巧上来讲：

1. 解题的时候要避免人脑递归，尽量去寻找问题的子问题，

2. 大部分的问题都有解题的模板代码，要记熟练，比如树的遍历、BFS、DFS、分治、回溯、二分查找、并查集、高级动态规划、各类排序算法、字符窜解法，等等。

3. 遇到短时间内想不出的题目要去看题解，多学习国际版的高票代码。

今年刚毕业出来，在校期间碌碌无为无亮点，简历直接被大厂刷掉，面试被问到成绩也会没了底气，不过算法训练营也让我更有信心了一点，希望做题这个习惯以后能够继续坚持下去，并且还是要多写学习总结，方便以后复习。
